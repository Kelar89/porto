Studi Kasus: Mengapa Saya Memperbaiki Server Dulu, Baru Menjalankan Iklan

Ditulis oleh Umar | 15 November 2025

← Kembali ke Semua Insights

Klien baru: "Anggaran Google Ads kami Rp 50 juta/bulan. Jalankan." Saya cek website mereka. Load time-nya 12 detik. Saya tolak pekerjaan itu.

Saya tidak menolaknya selamanya. Saya berkata, "Beri saya 3 hari untuk memperbaiki server Anda. Setelah itu, baru kita bicara anggaran iklan." Mereka bingung, tapi setuju.

Ini adalah kesalahan paling umum yang saya lihat: perusahaan rela "membakar uang" jutaan rupiah untuk iklan, tapi mengabaikan "fondasi" tempat iklan itu mendarat. Menjalankan iklan ke website yang lambat ibarat mengisi ember yang bocor.

Iklan Membeli 'Klik', Bukan 'Kesabaran'

Pikirkan alurnya. Pengguna melihat iklan Google Ads Anda yang menjanjikan. Mereka tertarik. Mereka klik.

Sekarang, jam pasir berputar. Google menemukan bahwa **53% pengguna mobile akan meninggalkan situs yang loading-nya lebih dari 3 detik.**

Website klien saya loading 12 detik. Ini berarti lebih dari separuh anggaran iklan mereka (Rp 25 juta+) terbuang hanya untuk membayar orang yang pergi sebelum halaman sempat terbuka. Marketer mereka menyalahkan bidding strategy. Saya menyalahkan server response time.

Diagnosa Hybrid Saya: IT Dulu, Ads Kemudian

Sebelum satu rupiah pun dihabiskan untuk iklan, saya melakukan ini:

  • Analisis TTFB (Time to First Byte): Server mereka butuh 4 detik hanya untuk merespons. Saya pindahkan hosting mereka dari shared hosting murah ke VPS yang terkonfigurasi dengan benar. TTFB turun menjadi 400ms.
  • Optimasi Aset (WordPress/CMS): Website mereka memuat 12 file CSS dan 15 file JavaScript. Saya meng-kombinasikan dan minify semuanya, mengurangi request dari 27 menjadi 4.
  • Konfigurasi Caching: Saya menginstal dan mengkonfigurasi server-level caching (Redis) dan browser caching. Pengguna yang kembali akan merasakan load time instan.

Hasil: Anggaran Sama, Konversi 2x Lipat

Setelah 3 hari, load time website turun dari 12 detik menjadi 2.1 detik. Bounce rate (angka pengguna yang langsung pergi) anjlok 45%.

Baru **setelah itu**, kami menyalakan anggaran iklan Rp 50 juta yang sama. Tanpa mengubah target audiens atau materi iklan sedikit pun, conversion rate mereka naik dua kali lipat di bulan pertama.

Jangan membayar untuk klik yang bounce. Jika Anda merasa konversi iklan Anda buruk, jangan buru-buru menyalahkan iklannya. Mungkin masalahnya ada pada "ember" Anda yang bocor.
